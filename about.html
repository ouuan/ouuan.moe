关于本博客

这是我继博客园、hexo-theme-next、hugo-theme-even 后的第四代博客。博客园和 Hexo 的文章已全部移至 Hugo 博客，但 Hugo 博客的文章（以 OI 相关内容为主）将不会移至本博客，可在 原 Hugo 博客 查看。

本博客由 ouuan 自己设计、制作，主要基于 îles（Vue、Vite）和 UnoCSS。设计有在一定程度上借鉴 hugo-theme-even、hexo-theme-next、Sukka's Blog、GitHub 的 Markdown 显示，但大部分是原创设计，且借鉴部分的代码基本都是自己写的。

博客正文部分使用的字体是 思源宋体，引用块部分使用的是 霞鹜文楷，二者均以 OFL 授权。web font 的划分、生成可以参考《另一种很新的中文字体网页嵌入方案》，其他排版优化（包括标点挤压的实现）详见《一些博客排版优化小 trick》。

代码高亮使用的是 Shiki，主题是 Night Owl（选了一个用的人较多，有暗色和亮色，配色整体偏蓝、和博客主题比较贴合的；我自己的 neovim 还是用的 gruvbox）。remark 插件和 Vue component 是自己写的，可以参考 remarkShiki.ts、CodeBlock.vue、CopyButton.vue（如果用的不是 iles 大概不能直接照抄）。

可以在遵循 AGPL v3（或其更新的版本）的条件下参考/使用 本博客代码 的 片段，但请 尽量不要 直接在整个代码的基础上进行修改，因为并没有按照人人可以拿来用的模板进行设计，很多地方会硬编码一些东西，修改比较困难，如果我这里有更新要同步也很麻烦，也可能在某些系统环境无法正常构建之类的。如果真的直接使用，也需要遵循 AGPL v3，并且请务必小心，一切后果自负。

本博客使用 ouuan 自行搭建的 Plausible Analytics 实例进行访问量统计，不使用 Cookies 且满足 GDPR 等条例要求。使用 giscus 提供评论服务，发表评论需要登录 GitHub 帐号，除了发表评论也可以给文章点赞/点踩。网站托管于 Cloudflare Pages。
